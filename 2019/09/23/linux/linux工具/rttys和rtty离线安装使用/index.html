客户端依赖

  • libev 高性能的事件循环库
  • libuwsc 一个轻量的针对嵌入式Linux的基于libev的WebSocket客户端C库。
  • mbedtls(polarssl)、CyaSSl(wolfssl)或者openssl 如果你需要支持SSL
